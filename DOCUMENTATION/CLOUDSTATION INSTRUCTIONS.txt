Method 2: Command Prompt (Text Diagram)
Open Command Prompt (type cmd in the Start Menu).
Navigate to the folder you want to diagram using the cd command (e.g., cd C:\Users\YourName\Documents).
Type tree and press Enter to see a simple folder structure.
Type tree /f and press Enter to include files in the diagram.
To save it, type tree /f > folder_structure.txt to create a text file.
